## Calculadora de Cambio de Moneda
Esta aplicación tiene la capacidad de realizar cambios de monedas en base a valores fijos.

### Principales Monedas
* Peso Dominicano [DOP]
* Dolar Estadounidense [USD]
* Euro Europeo [EUR]

### Simbologia
* Peso : _RD$_
* Dolar : _US$_
* Euro : _€_
